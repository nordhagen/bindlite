unbind()method 
public function unbind(target:Object, key:String = null):void

Removes a target from binding(s). Specifying a property to unbind is optional. Without a key argument, the target will be removed from all bindings, effectively releasing it from data binding all together.

Parameters

target:Object — The object to remove a binding from
 
key:String (default = null) — Optional predefined bindable property name. If specified, only this binding is removed.
